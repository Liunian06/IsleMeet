<h1 align="center">屿见 / IsleMeet</h1>

IsleMeet / 屿见 是一个 AI 陪伴类跨平台 APP，也是一个 AI-native 的陪伴系统。

项目采用递归式系统思路来组织体验：用户打开的是一个应用界面，实际承载的是一个可以持续扩展、持续陪伴、持续演进的 AI 系统。

## 项目目标

- 构建一个面向陪伴场景的跨平台应用
- 以 AI-native 的方式组织交互、状态和内容
- 保持简洁、克制、系统感强的产品体验
- 为后续的功能扩展、角色能力和系统能力预留清晰结构

## 技术方向

- `C++17`
- `Flutter UI`
- `SQLite`

## 当前状态

项目仍处于早期构建阶段，当前重点是完成基础架构、协作规范、UI 约束和后续核心功能的起步设计。

## 开发原则

- 先明确需求，再输出简明 PRD
- 以小步迭代推进实现
- 自主编译，并尽量进行真机验证
- 出现问题优先修复，再继续下一轮

## 设计方向

- 优先参考原生 iOS 的极简线条风格
- 保持简约、治愈、留白充足
- 强调内容、层级和系统感
- 避免玻璃磨砂、蓝紫色科技风和夸张装饰
